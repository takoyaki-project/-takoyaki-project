# 🐙 TAKOYAKI PROJECT

**小さな困りごとを、アプリにする。**

はじめまして、けいこです。
専業主婦歴13年、プログラミング未経験、パソコンは電源の入れ方から怪しかった超文系。
そんな私が、AIと一緒に「暮らし・学校・仕事のちょっと困った」を解決する小さなWebアプリを作っています。

- 🏠 開発環境はほぼ **iPhoneひとつ**
- 🤖 AIに相談しながら、仕様づくりからコードまで
- 📝 開発のリアルは note に書いています → [けいこ＠専業主婦歴13年からのアプリ開発](https://note.com/keiko_aibeginner)

---

## 📱 作ったもの

すべての作品は **[ポートフォリオサイト](https://takoyaki-project.github.io/)** から実際に触れます。

| アプリ | ひとこと | リポジトリ |
|---|---|---|
| 🪑 **席替えアプリ** | 名前入力不要・個人情報ゼロで使える学校向け席替えツール。隣席NG・前列指定・男女配置に対応。先生に使ってもらっています | [seat-assignment-app](https://github.com/takoyaki-project/seat-assignment-app) |
| 🛍️ **あれ売れた？** | バザー・文化祭・イベント販売の売上をワンタップ記録。おばあちゃん世代でも使える設計。PWA対応 | [are-ureta-event-register](https://github.com/takoyaki-project/are-ureta-event-register) |
| 🚪 **次行こか** | 訪問営業の「断られた」を記録して前を向くカウンター。営業現場を知る秘書が作りました | [courage-counter](https://github.com/takoyaki-project/courage-counter) |
| 💘 **AI男子〜恋も開発もバグだらけ！** | AIツールが恋愛シミュのキャラになったら？という遊び心全開のゲーム | [AI-danshi](https://github.com/takoyaki-project/AI-danshi) |
| 🗺️ **相続マップ** | 家族構成を入力すると相続関係図と法定相続分の目安がその場で描ける試作ツール（参考用） | ポートフォリオサイトから |
| 🎮 **たこやきクエスト** | ドラクエ風の開発日誌。冒険はまだ続く | [takoyaki-quest-log](https://github.com/takoyaki-project/takoyaki-quest-log) |

ほかにも、娘と一緒に作ったルーレットや家族用ミニゲームなど、小さな作品がいくつか。

---

## 🛠️ 作り方

**ChatGPT　Gemini で仕様を相談 → Claude でコード → CodePen / GitHub Pages で公開**

フレームワークより「誰が・いつ・どこで使うか」を先に考えるのが流儀です。
インストール不要・個人情報を集めない・スマホで完結、が基本方針。
